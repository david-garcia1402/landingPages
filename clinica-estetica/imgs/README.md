# Imagens da landing page

Coloque nesta pasta as imagens com os nomes abaixo para que a página exiba corretamente:

| Arquivo | Uso |
|---------|-----|
| `hero.jpg` | Banner principal da hero section |
| `profissional.jpg` | Foto da profissional (seção Sobre) |
| `botox.jpg` | Card do procedimento Botox |
| `preenchimento.jpg` | Card Preenchimento labial |
| `limpeza.jpg` | Card Limpeza de pele |
| `harmonizacao.jpg` | Card Harmonização facial |
| `corporais.jpg` | Card Tratamentos corporais |

Recomendação: imagens em formato JPG ou WebP, proporção 4:3 para os procedimentos, resolução mínima 800px na menor dimensão.
